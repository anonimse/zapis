# Telegram Bot для записи на прием в ВУЗы

Бот для онлайн записи на консультации в медицинские ВУЗы Санкт-Петербурга.

## Функционал

### Для пользователей:
- Выбор ВУЗа из списка
- Ввод личных данных (ФИО, телефон, Telegram)
- Выбор даты и времени приема
- Подтверждение записи
- Отмена записи

### Для администраторов:
- Просмотр всех записей по датам
- Просмотр занятых и свободных слотов
- Включение/выключение уведомлений о новых записях

## Структура проекта

- `bot.py` - основной файл бота
- `database.py` - работа с базой данных SQLite
- `config.py` - конфигурация (расписание, ВУЗы)
- `utils.py` - вспомогательные функции
- `.env` - переменные окружения (токен бота, админы)
- `requirements.txt` - зависимости Python
